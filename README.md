# Peewa

**Peewa** is a custom Discord bot designed to provide fun and utility commands for your server specialized for **Audition Next Gen** Game. 
It is built with **Node.js** and uses **discord.js** for connectivity with Discord’s API.

## 🚀 Features

Peewa currently includes:
- 🎵 Random song selection commands  
- 📊 Check you Character's Round Count command
- 🎲 Randomized selectors for games  
- 🎮 Simple interactive Discord slash commands  

> Peewa is a personal project and built for customization and growth.

## 💡 Prerequisites

Before running the bot, make sure you have:

- **Node.js** (v18+ recommended)
- A **Discord Bot Token** (create an app in the Discord Developer Portal)
- A **Server with Slash Commands enabled**

## 📦 Installation

```bash
git clone https://github.com/jgalvs/Peewa.git
cd Peewa
npm install
```
